# 模型支持情况

更新时间：2026-06-08
约束：本平台首期只围绕两类资源建模：Web Cookie/session 与 Agent Provider credential/profile。connector、sidecar、MCP-to-HTTP、订阅源或自托管服务只作为执行层或导入格式；不应替代账号资源本体。
数据来源：`media2api/provider_templates.py`、`media2api/catalog.py`、远端 `/v1/providers`、`/v1/accounts`、`/v1/model-mappings`。

## 1. 当前实现口径

| 事项 | 当前规则 |
|---|---|
| 上游第一方接口 | 禁用。管理台 provider onboarding 不再提供上游 `api_key` 或第一方 OAuth 获取入口。 |
| 上游第一方接口密钥 | 禁用。`/v1/admin/account-onboarding` 遇到 `auth_method=api_key` 会返回 `UPSTREAM_OFFICIAL_API_AUTH_NOT_ALLOWED`。 |
| 可接受账号材料 | 只暴露两类主路径：`web_cookie_provider + cookie_secret`、`agent_provider + agent_provider_credential`。旧的 OAuth/CLI/MCP/token/subscription 字段只能作为兼容导入材料，入库后必须归一到这两类。 |
| sub2api/subapi 类体验 | 用户选择平台后，导入本人已授权的 Web Cookie/session 或登记 Agent Provider profile；资源清单只用于批量导入这些资源。 |
| 本系统自己的调用密钥 | 保留。这是用户调用 `media2api` 的认证方式，不是上游模型接入凭据。 |

## 2. 额度来源分类

| 额度来源 | 含义 | 账号材料 |
|---|---|---|
| Web Cookie 额度 | 消耗用户已授权网页账号的会员次数、队列或生成权益 | `cookie_secret`、`secret://...`，执行器地址仅在项目要求时填写 |
| Agent Provider 额度 | 由 Codex、Gemini CLI、Qwen Code、Antigravity、MCP agent 等 runtime 消耗授权 profile | `agent_provider_credential`、`agent://...`，runtime endpoint 仅在项目要求时填写 |
| 第三方聚合额度 | 非首期核心；仅作为 fallback 或能力对照 | 必须包装成 Agent Provider 或执行层适配，不作为用户主表单的 token 资源 |
| 自建反代/自托管额度 | 非首期核心；可作为 Agent Provider 后端或执行层 | 只记录执行层配置，不替代 Web Cookie / Agent Provider 账号资源 |

| `image_to_image` | 图生图/变体 | OpenAI 图像连接器、Gemini、Qwen、即梦、Midjourney、Seedream、Flux |
| `image_edit` | 图片编辑 | OpenAI 图像连接器、Gemini、Qwen、Seededit、Flux/ControlNet |
| `text_to_video` | 文生视频 | Gemini/Veo、Grok、Qwen/Wan、即梦/Seedance、可灵、Luma、Runway、Pollinations、fal/Replicate |
| `image_to_video` | 图生视频 | Gemini/Veo、Grok、Qwen/Wan、即梦/Seedance、可灵、Luma、Runway、Pollinations、fal/Replicate |

## 9. 维护规则

1. 新增 provider 时必须说明它对应 Web Cookie 还是 Agent Provider；反代连接器或第三方聚合器只作为执行层/fallback 标记。
2. 新增账号时必须标记 `quota_source`：`web_cookie`、`agent_provider`、`third_party_fallback`、`self_hosted_runner`。
3. 不允许把上游第一方接口密钥、第一方 OAuth 长期令牌或第一方控制台链接作为用户接入教程。
4. 后台账号材料只按 Web Cookie/session 或 Agent Provider profile 两类主路径展示；旧订阅或引用格式必须最终归一到 `web_cookie_provider` 或 `agent_provider` 资源画像。
