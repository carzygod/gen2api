# 模型支持情况

更新时间：2026-06-08  
约束：本平台不接入、不引导、不保存任何上游第一方接口密钥或第一方调用凭据。所有上游模型资源必须通过反向代理类仓库、第三方聚合连接器、网页账号池连接器、Codex/ChatGPT agent 连接器或 Antigravity/Gemini agent 连接器接入。  
数据来源：`media2api/provider_templates.py`、`media2api/catalog.py`、远端 `/v1/providers`、`/v1/accounts`、`/v1/model-mappings`。

## 1. 当前实现口径

| 事项 | 当前规则 |
|---|---|
| 上游第一方接口 | 禁用。管理台 provider onboarding 不再提供上游 `api_key` 或第一方 OAuth 获取入口。 |
| 上游第一方接口密钥 | 禁用。`/v1/admin/account-onboarding` 遇到 `auth_method=api_key` 会返回 `UPSTREAM_OFFICIAL_API_AUTH_NOT_ALLOWED`。 |
| 可接受账号材料 | `token_reference`、`subscription_url`、`secret_json`、`vault://...`、`secret://...`。 |
| sub2api/subapi 类体验 | 用户选择平台后，到对应反代连接器后台复制订阅地址或账号引用，再粘贴到本系统。 |
| 本系统自己的调用密钥 | 保留。这是用户调用 `media2api` 的认证方式，不是上游模型接入凭据。 |

## 2. 额度来源分类

| 额度来源 | 含义 | 账号材料 |
|---|---|---|
| 网页反代额度 | 消耗上游网页账号的会员次数、队列或生成权益 | 连接器 Base URL + `token_reference` / `subscription_url` |
| 第三方聚合额度 | 消耗第三方聚合/转发平台账户余额 | 聚合连接器 Base URL + 连接器账号引用 |
| Codex/ChatGPT agent 额度 | 由 ChatGPT/Codex 图像 agent 或 sidecar 负责生成 | agent 连接器 Base URL + `vault://providers/openai_image/acct_01` |
| Antigravity/Gemini agent 额度 | 由 Antigravity/Gemini 浏览器/IDE agent 负责生成 | agent 连接器 Base URL + `vault://providers/gemini/acct_01` |
| 自建反代/自托管额度 | 消耗你自建 worker、GPU、队列或账号池资源 | 自建连接器 Base URL + 订阅/账号引用 |

## 3. 操作能力

| 操作 | 说明 | 典型平台 |
|---|---|---|
| `text_to_image` | 文生图 | OpenAI 图像连接器、Gemini/Nano Banana、Qwen、即梦、Midjourney、Pollinations、OpenRouter、fal/Replicate、Flux |
| `image_to_image` | 图生图/变体 | OpenAI 图像连接器、Gemini、Qwen、即梦、Midjourney、Seedream、Flux |
| `image_edit` | 图片编辑 | OpenAI 图像连接器、Gemini、Qwen、Seededit、Flux/ControlNet |
| `text_to_video` | 文生视频 | Gemini/Veo、Grok、Qwen/Wan、即梦/Seedance、可灵、Luma、Runway、Pollinations、fal/Replicate |
| `image_to_video` | 图生视频 | Gemini/Veo、Grok、Qwen/Wan、即梦/Seedance、可灵、Luma、Runway、Pollinations、fal/Replicate |
| `video_extend` | 视频续写/延展 | 可灵、Luma、Runway |

## 9. 维护规则

1. 新增 provider 时必须说明它对应哪个反代连接器或第三方聚合连接器。
2. 新增账号时必须标记 `quota_source`：`web_reverse`、`third_party_aggregator`、`codex_agent`、`antigravity_agent`、`self_hosted_proxy`。
3. 不允许把上游第一方接口密钥、第一方 OAuth 长期令牌或第一方控制台链接作为用户接入教程。
4. 后台账号材料只接受连接器订阅或引用：`token_reference`、`subscription_url`、`vault://...`、`secret://...`。
